Welcome to the Product Service Application

This is a simple web application designed to manage products. You can perform various operations such as creating, reading, updating, and deleting product records. The application is built using the Go programming language and the Gin framework.

The backend of this application handles product data stored in a JSON file. It provides RESTful APIs to interact with the product data. These APIs allow you to retrieve all products, fetch a product by its ID, add new products, update existing ones, and delete products.

This application provides a set of RESTful APIs to manage product data. Below is a summary of the available functions in the backend:

  • GetAllProducts: Retrieves all products and returns them as a JSON response.
  • GetProductById: Fetches a product by its unique ID and returns it as a JSON response.
  • CreateProduct: Adds a new product to the list and saves it to the JSON file.
  • UpdateProduct: Updates an existing product by its ID and saves the changes to the JSON file.
  • DeleteProduct: Deletes a product by its ID and updates the JSON file.
